# Marketing Playbook

The growth engine. Concrete, no fluff. **You** execute; the game already has the hooks (codes, "What's New," steal kill-feed).

## 1. Short-form clips (the #1 free lever)
Roblox UGC games grow on TikTok/Shorts/Reels. The **steal** is your money shot.
- **Format:** 9:16, 10–20s, loud hook in the first 0.5s. Screen-record the steal-POV: sneak onto a base → grab a glowing rare → run home → "+$" explodes. End on the rarest unit.
- **Caption/keywords:** "stealing the RAREST brainrot 🧠", game name in caption + bio link.
- **Cadence (realistic, solo):** 1–2 clips/day for the first 2 weeks, then 3–5/week. Volume + consistency beats polish. Repost winners across all 3 platforms.
- **What goes viral:** the biggest steal, the rarest pull, a rage-quit reaction, a "code expires tonight" urgency clip.

## 2. Discord (retention + feedback)
- Channels: `#announcements` (codes + updates), `#codes`, `#trading`, `#bugs`, `#clips` (UGC from players), `#general`.
- Put the invite in the experience description + a HUD link later. Drop each new code in `#announcements` the moment the update ships.
- Run simple events: "first to rebirth 5x," "rarest collection screenshot." Reward with… a code.

## 3. Small-creator outreach
- DM 20–50 creators with **10k–100k** subs in the Roblox/brainrot niche (big creators ignore you; small ones convert). Offer: a free private server, early code, or a unit named after them later.
- Give them a unique code (just add it to `CodesConfig.lua`) so you can see which creator drove redemptions.

## 4. Update cadence (what the algorithm rewards)
Roblox surfaces games that **update often + retain**. Ship **weekly** early on:
- Every update: 1–3 new brainrots (just edit `Catalog.lua`), 1 new **code** (`CodesConfig.lua`), a balance tweak, bump `GameInfo.Version` (re-shows "What's New" with the code).
- Announce in Discord + a clip + the experience update notes. The codes ARE the reason players return — never ship an update without one.

## 5. Paid ads (LATER — only after retention is proven)
- Don't pay for traffic into a leaky bucket. First get **D1 retention > ~25%** and a watchable session (check analytics).
- Then test Roblox's **Sponsored/Ad Manager** with a small daily budget, optimizing for the icon with the best organic CTR. Scale only what's profitable (track Robux/visit vs cost).

## Priority order
1. Nail the icon + name (CTR). 2. Clip daily. 3. Weekly updates + codes. 4. Discord. 5. Outreach. 6. (much later) paid ads.
